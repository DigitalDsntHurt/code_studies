Experiment with `border` property

use only `border: solid`

use only `border: 1px`

use only `border: black`

okay, so if we want to see something, `border: ` definitely needs at least a `solid` argument

It turns out that `border` is actually shorthand for three different css properties:

  • border-style
  • border-width
  • border-color

Here are a few different values we can provide to border-style :

solid
dotted
dashed
groove
double
